SimpleLogin

如果用一个邮箱来注册所有账号，电子邮件地址可能会暴露给垃圾邮件发送者

因此，我们可以使用 SimpleLogin 来保护我们真实的邮箱地址

关于 SimpleLogin

官网：SimpleLogin.io

SimpleLogin 是一个开源项目：SimpleLogin · GitHub

SimpleLogin 提供电子邮件的转发服务，我们可以生成随机的永久邮箱名使用，它将收到的邮件转发到我们真实的邮箱，从而保护了我们的邮箱不被暴露

当然，它也支持从别名发送邮件

使用过程中我们可以对所有别名进行管理，无论是新建别名、查看每个别名使用情况以及删除别名等都非常方便

并且 SimpleLogin 还提供多种域名可供选择，同时还支持使用自己的域名等，满足了个性化的需求

方案

假设我们使用的邮箱是 real@mail.com，SimpleLogin 提供的邮箱是 anything@simplelogin.com

  • 用 real@mail.com 去注册 SimpleLogin

  • 在每次需要使用邮箱注册账号的时候，根据注册的平台生成不同的邮箱名

    如注册 Apple ID 时使用 AppleID@simplelogin.com，注册 GitHub 时使用 GitHub@simplelogin.com，以此类推

  • SimpleLogin 将这些邮箱收到的邮件统一转发给 real@mail.com

这样，我们就可以精确地对每一个服务的邮件进行管理，不仅把自己的 real@mail.com 隐藏了起来，而且当我们发现某个服务的邮箱如 abc@SimpleLogin.com 被泄漏，收到了并非来自 abc 的垃圾邮件，也可以及时地将 abc@simplelogin.com 停用，然后把 abc 的账号换绑为新的邮箱

其中，如果设置了自定义域名就可以直接使用 anything@domain.com，第一次收到邮件后会自动创建别名，免去了每次要使用前手动创建别名的麻烦，就跟使用普通邮箱的便捷度差不多

使用

注册

  1. 前往 SimpleLogin 官网，点击右上角 Sign Up 注册账号

  2. 前往邮箱，点击确认邮箱完成注册

  3. 前往控制面板，可以在这里对所有的邮箱进行管理

  • 新注册账户有一周的 Premium 免费体验期

新建别名

  1. 点击 + New Custom Alias 新建一个自定义的别名

  2. 输入别名前缀，选择一个域名，点击 Create 完成新建

    • 每次会随机生成别名后缀，与前缀之间以 . 分隔，新建的邮箱如 anything.random@simplelogin.com

  3. 也可以在控制面板点击 Random Alias 快速新建一个别名

管理

在控制面板可以对所有的别名进行管理

  • 停用/启用

    点击别名右边的开关

  • 删除

    点击 More，Delete

  • 查看别名用量

    点击 More

  • 查看往来邮箱

    点击 Contacts

  • 拦截发信者

    1. 点击 Contacts
    2. 点击发信者右边开关

  • 以别名发送邮件

    1. 点击 Contacts
    2. 点击 Create reverse-alias
    3. 使用自己的邮箱发送邮件到 reverse-alias

  • ……

创建子域名

  1. 在控制面板点击 Subdomains

  2. 输入自定义子域名，选择根域名，点击 Create 完成创建

    • 生成的域名如 subdomain.simplelogin.com
    • 用于创建邮箱如 anything@subdomain.simplelogin.com

编辑收件箱

  1. 在控制面板点击 Mailboxes

  2. 可以更改收件邮箱或者新建更多的收件邮箱，对于每个别名可多选，转发给多个收件邮箱

使用自定义域名

  1. 在控制面板点击 Domains

  2. 输入域名，点击 Create

    • 可以是二级域名，如拥有 domain.com，可以使用 subdomain.domain.com

  3. 前往域名管理网站，按要求添加 TXT 记录，验证域名所有权

    • 修改需要等待一段时间生效

  4. 完成验证后，继续按要求添加 MX 记录

  5. 建议按提示完成 SPF、DKIM 及 DMARC 设置

  6. 配置完成后点击 Details 进入域名管理页面

    • Auto create/on the fly alias

      建议打开 Catch All 选项，打开后所有发送到该域名的邮件都会自动创建别名并转发到收件邮箱

    • 如果觉得将所有收到的邮件都接收会导致收到垃圾邮件，可以点击右侧 Auto Create，输入正则表达式，这样就只接收发送到符合规则的邮箱的邮件

自定义目录

就是低配版的 Auto create/on the fly alias，设置自定义字符串为前缀，所有发到以这个字符串为前缀的邮件都将自动创建别名并转发到收件箱

  1. 在控制面板点击 Directories
  2. 输入目录名，点击 Create，需未被使用

升级

以上有部分功能是需要 Premium 才可以使用，可以按照以下方式升级，$30/年的价格也比较良心

  1. 点击右上角的 Upgrade

  2. 支持使用银行卡、PayPal 及加密货币支付
